# Guía de Contribución

Gracias por tu interés en contribuir a este proyecto de BLIST.

1. Haz un fork del repositorio.
2. Crea una rama para tu funcionalidad (`git checkout -b feature/nombre-funcionalidad`).
3. Realiza tus cambios y verifica que el código cumpla con los estándares.
4. Envía un Pull Request para revisión.

¡Tu colaboración ayuda a evolucionar nuestra entidad!
